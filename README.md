## Escape codes

| code | meaning         |
| ---- | --------------- |
| \a   | audible alert   |
| \b   | backspace       |
| \f   | form feed       |
| \n   | newline         |
| \r   | carriage return |
| \t   | tab             |
| \v   | vertical tab    |

## Format Specifiers

| %   | meaning                        |
| --- | ------------------------------ |
| %c  | character                      |
| %d  | decimal integer(base10) number |
| %e  | exponential floating-point     |
| %f  | floating-point number          |
| %i  | integer number                 |
| %o  | octal(base8) number            |
